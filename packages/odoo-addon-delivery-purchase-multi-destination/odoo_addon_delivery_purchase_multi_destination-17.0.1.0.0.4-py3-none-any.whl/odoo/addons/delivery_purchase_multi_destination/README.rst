This is a glue module for using multi destinations for purchase
sendings.

Configuration
=============

To configure this module please refer to configure section of the
delivery_multi_destination module.

Usage
=====

1. When using the delivery method in a Purchase order, provider address
   will be used for computing the delivery price according introduced
   provider.
